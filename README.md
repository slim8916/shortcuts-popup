# My GNOME Shortcut Pop-up Extension

A simple GNOME Shell extension that displays a pop-up window with a user-defined list of keyboard shortcuts and their descriptions.

## Installation

1. Clone this repo into `~/.local/share/gnome-shell/extensions/`:

   ```bash
   git clone https://github.com/slim8916/gnome-shortcuts-popup
   ```
2. Reload GNOME Shell: Press <kbd>Alt</kbd>+<kbd>F2</kbd>, type `r`, and hit <kbd>Enter</kbd>.
3. Enable the extension with [GNOME Tweaks](https://wiki.gnome.org/Apps/Tweaks).

## Usage

* Open the pop-up window by pressing <kbd>Ctrl</kbd>+<kbd>Alt</kbd>+<kbd>Shift</kbd>+<kbd>O</kbd>, or by clicking the extension's icon in the GNOME bar.
* Add shortcuts either via the extensionâ€™s settings dialog or by uploading a JSON file. An example structure is provided in model.json in the extension folder.
